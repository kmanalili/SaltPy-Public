**SaltPy** is an open-source Python toolkit for processing, analyzing, and visualizing sonar and geospatial data from solution-mined salt caverns.

## Installation

Install from PyPI:

```bash
pip install saltpy
```

Optional dependency for geomagnetic calculations:

```bash
pip install git+https://github.com/pleiszenburg/pyCRGI.git@master
pip install saltpy
```
